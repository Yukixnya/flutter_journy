# journal_app

## Create a Flutter Journal application where users can:
- Add journal entries
- View a list of entries

## Each journal entry should contain:
- Title
- Description/content
- Date or timestamp

## Use appropriate widgets:
- TextField
- ListView
- Card

### Manage state properly to add and display entries.

### Ensure data persists during app runtime (in-memory is acceptable unless specified).

### Apps should handle empty states gracefully.
